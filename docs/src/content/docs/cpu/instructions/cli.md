title: CLI
---

Esta instrucción inhabilita las [interrupciones](/cpu/#interrupciones) por hardware, cambiando la [_flag_](/cpu/#flags) `IF=0`.

## Uso

```vonsim
CLI
```

## Codificación

`00011000`
